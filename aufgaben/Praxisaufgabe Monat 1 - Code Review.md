📋 Die Aufgabe
Du erhältst eine PHP-Klasse OrderCalculator.php, die Bestellberechnungen für einen Online-Shop durchführt. Die Klasse wurde bereits in Produktion eingesetzt, aber es gab vereinzelt Beschwerden von Kunden über falsche Berechnungen.
Deine Aufgabe:
Führe ein Code Review durch und finde so viele Fehler wie möglich!
Dauer: 25 Minuten
Methode: Manuelle Code-Analyse ohne Ausführung

🔍 Vorgehen

Öffne die Datei OrderCalculator.php
Untersuche den Code Zeile für Zeile auf potenzielle Fehler
Notiere jeden gefundenen Fehler:

Zeile
Was ist falsch?
Warum ist es ein Problem?



Beispiel-Notation:
Zeile 23: Division durch Null nicht abgefangen
Problem: Wenn $quantity = 0, gibt es einen Fatal Error

⚠️ Wichtige Hinweise

❌ Nicht den Code ausführen - Wir üben statisches Testen
❌ Nicht im Internet nach Lösungen suchen
✅ Bei Unsicherheit trotzdem notieren
✅ Es geht nicht darum, ALLE Fehler zu finden


📁 Benötigte Dateien

OrderCalculator.php
Notizblock oder Editor für deine Notizen
